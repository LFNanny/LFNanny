# 🎨 LF Nanny — Brand Guidelines (V1)

Este documento define las normas de uso de la identidad visual de **LF Nanny**.
Su objetivo es garantizar **coherencia de marca** en web, documentación, producto y aplicación móvil.

---

## 🟡 Logotipos

LF Nanny dispone de **dos logotipos oficiales**, con usos claramente diferenciados.

### ✅ Logo A — Logotipo principal (corporativo)

**Archivo:**
- `LF-Nanny-Logo.png`

**Uso autorizado:**
- Web (landing, blog, legal, etc.)
- Documentación (PRD, Brand Kit, presentaciones)
- Comunicaciones (emails, PDFs, materiales para socios)
- Material corporativo

**Normas:**
- Es el **único logotipo corporativo**
- No debe recortarse ni deformarse
- No usar como icono de aplicación

---

### ✅ Logo B — Icono de aplicación (iOS / Android)

**Origen:**
- Derivado del Logo A
- Adaptado específicamente como icono de app

**Uso exclusivo:**
- Icono de la aplicación móvil
- Elementos de interfaz asociados a la app

**NO usar:**
- En web
- En documentación corporativa
- En comunicaciones externas

## 🎨 Colores corporativos

| Color               | HEX     | Uso principal                         |
|--------------------|---------|----------------------------------------|
| Amarillo principal | #F4C430 | Accentos, llamadas a la acción (CTA)   |
| Azul principal     | #264653 | Base de marca, titulares, confianza    |
| Texto              | #1F2937 | Texto principal                        |

**Normas:**
- No introducir colores nuevos sin revisión
- Mantener contraste suficiente para accesibilidad

---

## 🚫 Usos no permitidos

- Usar el Logo B como logotipo corporativo
- Mezclar Logo A y Logo B en la misma pieza
- Cambiar colores oficiales
- Aplicar sombras, degradados o efectos no definidos
- Usar versiones descargadas de terceros
